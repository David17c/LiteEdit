# LiteEdit

A very basic gui text editor written in python.

| Keybind | Action |
|--------------|--------|
| CTRL + o | Open file |
| CTRL + s | Save file |
| CTRL + n | New file |
| CTRL + q | Quit |
| CTRL + z | Undo |
| CTRL + y | Redo |
| CTRL + a | Select all |
| CTRL + c | Copy selected |
| CTRL + v | Paste |

LiteEdit actively being developed and not currently meant to be used contributions are welcome
