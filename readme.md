# LiteEdit

A basic gui text editor written in python.

| Keybind  | Action        
|----------|---------------
| CTRL + o | Open file     
| CTRL + s | Save file     
| CTRL + n | New file      
| CTRL + q | Quit          
| CTRL + z | Undo          
| CTRL + y | Redo          
| CTRL + a | Select all    
| CTRL + c | Copy selected 
| CTRL + v | Paste         

LiteEdit is actively being developed and not yet ready for "real" use. Contributions are welcome.
