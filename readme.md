# LiteEdit

Lightweight text-editor written in Python

| Keybind  | Action              
|----------|---------------------
| CTRL + o | Open file           
| CTRL + s | Save file           
| CTRL + n | New file            
| CTRL + q | Quit                
| CTRL + z | Undo                
| CTRL + y | Redo                
| CTRL + a | Select all          
| CTRL + l | select current line 
| CTRL + c | Copy selected       
| CTRL + v | Paste
